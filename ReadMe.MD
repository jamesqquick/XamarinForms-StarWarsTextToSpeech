**Star Wars Text To Speech**
======
DependencyService example that builds upon the Star Wars Image Gallery sample to implement text to speech of the character names.

## **How to Use**

Clone the Xamarin Forms samples with the following command.

```$ git clone https://github.com/jamesqquick/XamarinForms-StarWarsTextToSpeech.git ```

Then, open up the solution file (.sln) in either Visual Studio or Xamarin Studio to run the sample.

## **Technology**

 2. Visual Studio
 3. Xamarin Studio
 4. Android, IOS, and Windows
